Finnish Meteorological Institute open data added to rOpenGov

FMI package for Finnish Meteorological Institute

An R package fmi for the Finnish Meteorological Institute open data API has been released. The package provides an access from R to many data sets including weather conditions, climate scenarios, sea related observations and sun radiation in Finland. Installation instructions, examples and other details are provided in the tutorial. The package is in beta and test experiences and contributions are welcome! For the contact details, see the package home page. The package has been jointly developed with rwfs package, which provides a generic access to WFS interfaces in R.

Note Oct 10, 2020: the package is now fmi2
